# Code of Conduct

This project adopts the **Contributor Covenant v2.1** without modification.

The full text is published at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

A plain-text copy is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md

## Scope

This Code of Conduct applies within all project spaces — the repository, issues, pull requests, discussions, and any community channel that identifies as part of the icuvisor project — and also applies when an individual is officially representing the project in public spaces.

## Reporting

To report a concern, email **conduct@icuvisor.dev** (or, until that mailbox is provisioned, the address listed in [SECURITY.md](SECURITY.md)). Reports are handled confidentially by the project maintainers.

## Enforcement

Maintainers will follow the Contributor Covenant's [Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) when determining the response to any reported incident.
